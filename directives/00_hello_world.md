# Hello World Directive

## Goal
Verify that the agent system is correctly configured and can execute Python scripts.

## Inputs
- None

## Tools
- `execution/hello_world.py`

## Instructions
1. Run `execution/hello_world.py`.
2. Verify that the output is "Hello from Antigravity!".

## Edge Cases
- If the script fails, check that Python is installed and the `execution` directory is in the path.
